## 概要

エンジニアとしての成長過程を記録し、外部に発信するためのポートフォリオサイト「**Iterate.log**」を個人開発しました。

「反復（Iterate）」と「記録（.log）」を掛け合わせた名前の通り、日々の学びを積み重ねていくことをコンセプトとしています。

## 技術構成

| 技術 | 用途 |
|------|------|
| **Python (Flask)** | バックエンド・ルーティング |
| **Jinja2** | テンプレートエンジン |
| **Markdown + YAML Front Matter** | コンテンツ管理 |
| **Vanilla CSS** | スタイリング（ホワイト・ミニマルデザイン） |
| **Vercel** | ホスティング・デプロイ |

## 主な機能

- **日次振り返り**: 毎日の学習内容を Markdown で記録
- **月次振り返り**: 月ごとの成長をまとめて振り返り
- **プロフィール**: 経歴・スキル・タイムラインの表示
- **実績ページ**: 過去のプロジェクトをカード形式で一覧表示

## デザインコンセプト

- **ホワイト・ミニマル**: 白を基調とした清潔感のあるデザイン
- **余白の活用**: 広い余白でコンテンツの可読性を向上
- **控えめなアニメーション**: ホバー時の浮き上がりなど、さりげない演出
